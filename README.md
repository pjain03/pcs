# COMP112: Networks, Final Project

## Compilation
Run the following commands:
1. `cd src`
2. `chmod u+x compile`
3. `./compile`

## Usage
1. Run the following commands:
    * `chmod u+x server`
    * `chmod u+x client`
2. Run the server using:
    * `./server <port number>`
3. Edit the `client` file to include the server's address. Then run the client using:
    * `./client <port number>`
